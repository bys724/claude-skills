# Paper Summary Skill

Interactive paper reading and personalized summarization workflow for researchers.

## Files

- `skill.md` - Main skill documentation (English, for Claude)
- `../../paper_summary_guide.md` - Original guide (Korean, legacy)

## Purpose

Helps researchers:
1. Read and discuss papers interactively with Claude
2. Generate personalized Korean summaries reflecting their perspective
3. Build a searchable network of paper relationships
4. Organize papers with categorized tags

## Usage

When working with this skill, Claude will:
- Read the provided paper
- Engage in discussion about the content
- Create a customized summary based on the conversation
- Suggest related papers when appropriate

See `skill.md` for detailed workflow and guidelines.
